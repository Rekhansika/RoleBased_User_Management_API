# RoleBased_User_Management_API

## Project Overview
This project is a **Role-Based User Management API** built using **Node.js, Express.js, and MongoDB**.  
It implements **JWT authentication and role-based authorization** to control access to API endpoints.

The system allows **admin users to manage other users**, including blocking users and promoting them to admin roles.  
An **audit log** is maintained to track important admin actions.

---

## Features

- User Registration
- User Login with JWT Authentication
- Role-Based Access Control (Admin / User)
- Admin Protected Routes
- Block Users
- Promote Users to Admin
- Audit Logging for Admin Actions
- MongoDB Database Integration

---
